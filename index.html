Multi-User Blog

The Multi-User Blog Web Application is a Python application that implements a web-based multi-user blog. Users can post information to the blog and also comment on other user's posts. Users can register with a unique user name and password. The application is written to be hosted on the Google Cloud Platform using the Google App Engine. This application was created as partial fulfillment of the Udacity Full Stack Web Developer Nanodegree. Specifically, it is Project 3: Multi-User Blog.

Tournament Planner

The Tournament Planner Application is a Python application that keeps track of players and matches in a Swiss-system tournament. The application uses a PostgreSQL database to keep track of the players and matches in the tournament. This application was created as partial fulfillment of the Udacity Full Stack Web Developer Nanodegree. Specifically, it is Project 4: Tournament Results.

Movie Trailer Website

The Entertainment Centre Application is a Python application where users can see the poster image of their favourite movies and watch the trailers. The application stores a list of movie titles, poster images and movie trailer URLs which are rendered on a web page. This application was created as partial fulfillment of the Udacity Full Stack Web Developer Nanodegree. Specifically, it is Project 1: Movie Trailer Website.
